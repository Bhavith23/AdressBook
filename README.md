# AdressBook
Developed an Address Book application in C to efficiently manage contact information. The application provides functionalities to store, search, edit, delete, and list contacts using file handling and structured data management techniques.

Key Features:
- Add new contacts
- Search contacts by name, phone number, or email
- Edit existing contact information
- Delete contacts
- Display all saved contacts
- Data persistence using file handling
- Input validation for contact details
- Modular and maintainable code structure

Technologies Used:
- C Programming
- Structures
- File Handling
- Dynamic Memory Management
- Modular Programming

Learning Outcomes:
- Practical implementation of structures and file operations
- Data management and validation techniques
- Modular software design principles
- Debugging and testing of C applications
- Version control using Git and GitHub

This project demonstrates fundamental software development concepts, including data storage, retrieval, manipulation, and user interaction through a menu-driven interface.
